Este proyecto implementa un sistema logístico de gestión de inventario utilizando Estructuras de Datos Puras en Python. Para demostrar el dominio algorítmico, la arquitectura prescinde de diccionarios (Tablas Hash) para la red principal, basándose en su lugar en modelos matemáticos espaciales.Arquitectura Técnica:Grafos (Lista de Adyacencia): El almacén se modela como un grafo no dirigido. Cada estantería actúa como un vértice (Nodo) y las conexiones físicas entre ellas (pasillos) actúan como aristas. Esto aporta una validación topológica real: no se pueden transferir productos entre zonas que no estén conectadas.Listas Enlazadas: El inventario interno de cada estantería se gestiona mediante una lista enlazada simple. Esto garantiza una eficiencia de inserción de $\Omega(1)$ al añadir nuevos productos por la cabecera, optimizando el uso de memoria en entornos de alta rotación.Algoritmo BFS (Búsqueda en Anchura): Para la localización de productos, se ha implementado un recorrido BFS apoyado por una Cola FIFO (deque). En un entorno logístico, esto asegura que el sistema explore primero las estanterías más cercanas al origen antes de ramificarse hacia pasillos más lejanos.
